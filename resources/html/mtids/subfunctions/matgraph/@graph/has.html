has

PURPOSE

has --- check if the graph has a particular vertex or edge

SYNOPSIS

function yn = has(g,u,v,dir)

DESCRIPTION 

 has --- check if the graph has a particular vertex or edge
 has(g,u) --- check if g contains the vertex u
 has(g,u,v) --- check if g contains the edge uv
 has(g,u,v,dir) -- treats 'g' as directed
