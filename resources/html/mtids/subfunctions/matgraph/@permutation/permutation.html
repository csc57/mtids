permutation

PURPOSE

constructor for the permutation class

SYNOPSIS

DESCRIPTION 

 constructor for the permutation class
 p = permutation(n) --- create a new (identity) permutation on 1:n
 p = permutation(vec) --- create a new permutation based on the array vec
 (which should be a permutation of 1:n).
