any_matrix_to_elist

PURPOSE

ANY_MATRIX_TO_ELIST Reshapes an adjacency matrix into an edge list

DESCRIPTION 

ANY_MATRIX_TO_ELIST Reshapes an adjacency matrix into an edge list

 This function takes a matrix and outputs an edge list
