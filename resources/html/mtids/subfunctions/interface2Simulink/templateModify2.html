templateModify2

PURPOSE

TEMPLATEMODIFY2 adapts the template according to given input

DESCRIPTION 

TEMPLATEMODIFY2 adapts the template according to given input

 INPUT:    nodeNumber      -- Index of node in MTIDS

 OUTPUT:   (none)          -- Changes are written directly into the open model
