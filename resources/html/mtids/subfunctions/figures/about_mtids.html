DESCRIPTION 

ABOUT_MTIDS M-file for about_mtids.fig
      ABOUT_MTIDS by itself, creates a new ABOUT_MTIDS or raises the
      existing singleton*.

      H = ABOUT_MTIDS returns the handle to a new ABOUT_MTIDS or the handle to
      the existing singleton*.

      ABOUT_MTIDS('CALLBACK',hObject,eventData,handles,...) calls the local
      function named CALLBACK in ABOUT_MTIDS.M with the given input arguments.

      ABOUT_MTIDS('Property','Value',...) creates a new ABOUT_MTIDS or raises the
      existing singleton*.  Starting from the left, property value pairs are
      applied to the GUI before about_mtids_OpeningFcn gets called.  An
      unrecognized property name or invalid value makes property application
      stop.  All inputs are passed to about_mtids_OpeningFcn via varargin.

      *See GUI Options on GUIDE's Tools menu.  Choose "GUI allows only one
      instance to run (singleton)".
